# CRUD application

1. First of all, you've to set up your database connections. 
Install` 
  MySql Command line prompt, 
  Mysql Workbench (my version is 8.0),
  MySql Server. 
In params my user is 'root' and password is '12345678'.
2. There are not included npm packages, u should run them with npm i in both folders.
3. Run the server index.js file in the Server folder on 'localhost:8080' port via command npm run server. 
4. At last, execute files for frontend development in the Client folder via command npm run init.
